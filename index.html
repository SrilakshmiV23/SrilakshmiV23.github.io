Portfolio

Enterprise Project — Allianz Arcnet Kafka Messaging Platform

At Allianz, I worked on the Arcnet Kafka messaging platform that supported communication between distributed enterprise backend services using event-driven microservices architecture. Apache Kafka enabled asynchronous communication between applications, improving scalability and ensuring reliable data flow across services.

Within this architecture, Kafka producers published events generated by backend applications while Kafka consumers processed events and triggered downstream workflows. This improved system responsiveness and supported reliable enterprise-level communication between services.

My responsibilities included supporting Kafka producer and consumer integration, assisting REST API communication between services, validating message workflows, supporting retry logic and error-handling mechanisms, debugging messaging pipelines, and resolving production-level backend issues.

During my undergraduate studies, I developed a Machine Learning-based system that classified teaching styles from lecture video datasets. The project involved preprocessing video data, extracting meaningful visual features from frames, and applying classification algorithms to identify instructional patterns and categorize teaching approaches.

This project strengthened my understanding of dataset preparation, feature extraction, supervised learning models, and evaluation techniques used in real-world multimedia classification systems.
