At Allianz, I worked on the Arcnet Kafka messaging system project which enabled real-time communication between distributed enterprise services.

In large-scale enterprise applications, multiple backend services must exchange data reliably without depending on direct connections. Apache Kafka enables this through asynchronous event-driven messaging where producers publish messages and consumers process them independently.

In this project, I worked on implementing Kafka producers and consumers, supporting REST API integrations, debugging messaging workflows, assisting testing activities, and improving system stability.
