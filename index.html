Portfolio

Enterprise Project — Allianz Arcnet Kafka Messaging Platform

At Tata Consultancy Services, I worked on the Allianz Arcnet Kafka messaging platform that supported communication between distributed enterprise backend services using event-driven microservices architecture.

Kafka producers generated backend service events while Kafka consumers processed those events to trigger downstream workflows such as validation pipelines, service coordination logic, and business-rule execution processes. This architecture improved scalability and reduced dependencies between services.

My responsibilities included supporting Kafka producer and consumer integrations, assisting REST API communication between services, validating message-processing workflows, supporting retry and error-handling mechanisms, and debugging production messaging pipelines in enterprise environments.

Machine Learning Project — Video Classification Based on Teaching Style

During my undergraduate studies, I developed a machine learning-based system that classified teaching styles from lecture video datasets using feature extraction techniques and supervised learning classification models.

The project involved preprocessing video datasets, extracting frame-level visual features, training classification models, and evaluating prediction performance to understand instructional behavior patterns in multimedia datasets.
